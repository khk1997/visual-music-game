# visual-music-game

Live demo: https://khk1997.github.io/visual-music-game/

A browser-based music visual playground built with HTML, Tone.js, and Three.js.

## Play

Open the live demo link above, then use your keyboard to trigger notes and visuals.

## Development

To update the site after local changes:

```bash
git add .
git commit -m "update"
git push origin main
```
